随着低空经济、无人机巡检和空地协同感知系统的发展，车辆检索的应用场景正在从传统地面摄像头网络扩展到无人机与地面平台之间的跨视角协同。与常规车辆重识别任务不同，空地车辆检索要求模型在无人机俯视或斜俯视图像与地面近景图像之间建立同一车辆的身份关联。这一任务在交通监管、园区巡逻、应急搜寻、港口调度和复杂地形巡检中具有明确需求：无人机可以在大范围内发现目标车辆，地面摄像头或移动端设备则需要进一步完成确认、回溯或拦截。

然而，空地车辆检索面临比传统车辆 Re-ID 更强的视角不对称问题。地面视角通常能够观察车辆的车头、车尾、侧面、车窗、轮毂和局部纹理，而无人机视角更多呈现车顶、天窗、整体轮廓和停放姿态。两类视角中可见部件高度不一致，真正可对齐的细粒度区域非常有限。同时，无人机图像常伴随小目标、低分辨率、遮挡、强透视形变和复杂背景干扰，使模型容易依赖停车位、道路纹理、邻车布局或相机域特征，而不是学习车辆本身稳定的跨视角身份线索。

现有车辆 Re-ID 数据集大多建立在地面多摄像头网络上，难以充分覆盖 UAV-to-ground 的跨平台匹配问题。以 CVnet/CVPair 为代表的真实空地车辆数据提供了 aerial-to-ground 与 ground-to-aerial 双向检索协议，为研究强视角差异下的车辆身份匹配奠定了重要基础。但这类数据仍主要以图像和身份标签为核心，缺少自然语言描述、结构化属性、样本不确定性和可审计质量标记。因此，它们难以支持更贴近真实人机协同场景的语言查询、属性检索、文本引导跨视角检索和模型失败原因分析。

在真实应用中，查询往往并不总是一张清晰、完整的车辆图像。操作者更可能输入“白色 SUV”“车顶带天窗的深色轿车”“停车场边缘的厢式货车”等语言描述，或先根据颜色、车型、车身结构进行粗筛，再由人工或更强模型完成实例确认。尤其在车牌不可见、被遮挡、因隐私合规被脱敏，或无人机图像目标过小的情况下，单纯依赖视觉特征进行精确实例匹配会受到明显限制。此时，颜色、车型、车顶结构、车身轮廓、可见部件和遮挡状态等语义属性，可以为跨视角检索提供更稳定、可解释的中间表示。

为此，本文提出 TAG-VR（Text-enhanced Aerial-Ground Vehicle Retrieval），一个面向空地车辆检索的文本增强 benchmark。TAG-VR 以 CVnet/CVPair 真实空地同 ID 数据为核心，并进一步引入经过处理的空地跟踪与目标识别数据集作为多源扩展。对于这类数据源，本文从原始空地协同场景中筛选车辆目标，抽取无人机视角与地面视角下的车辆图像对，并结合轨迹标注、时空同步关系、检测框、平台视角信息和人工复核建立可靠的车辆关联。随后，本文为这些配对图像补充统一的自然语言描述、结构化属性和质量控制信息，使其能够与 CVnet/CVPair 共同构成多源文本增强空地车辆检索数据集。

与普通图像 caption 数据不同，TAG-VR 的文本标注并不追求对每张图像生成自由描述，而是强调跨视角稳定身份语义的抽取。具体而言，本文构建两层标注：ID 级描述记录车辆在不同视角下相对稳定的颜色、粗粒度车型、车身轮廓和显著结构；图像级属性记录当前图像中实际可见的方向、部件、遮挡、目标尺度、背景干扰和不确定性。背景信息被显式记录用于错误分析，但不作为车辆身份特征，从而使背景泄漏和视角捷径从隐性风险转化为可分析、可评估的问题。此外，TAG-VR 引入可审计的 VLM 辅助标注流程。每个车辆 ID 生成稳定的身份级描述，每张图像记录当前视角下实际可见的属性、不确定项和 QA 状态；对于小目标、遮挡、颜色难辨或目标边界不稳定的样本，允许标记为 uncertain 或进入人工复核流程。该机制不仅降低了大规模文本标注成本，也避免了视觉语言模型在低质量样本上强行生成品牌、型号、车牌或不可见细节等幻觉信息，使数据集更适合作为严肃 benchmark 使用。基于上述设计，TAG-VR 支持多种任务设置，包括传统 visual a2g/g2a Re-ID、text-to-ground retrieval、text-to-UAV retrieval、image-to-text retrieval、text-guided cross-view retrieval、attribute/type retrieval等。通过这些任务，本文希望将空地车辆检索从单一图像匹配问题扩展为融合视觉、语言、属性和质量控制的多模态 benchmark，从而推动更鲁棒、更可解释的人机协同检索方法研究。

基于TAG-VR数据集，我们提出了一个面向空地跨视角图文对齐的 baseline 方法（暂记为 [Baseline Name]，现在还不确定名字和技术路线）。该方法旨在验证 TAG-VR 不仅能够支持传统视觉 Re-ID，也能够支持文本增强的跨视角车辆检索。与直接使用通用图文模型进行零样本匹配不同，[Baseline Name] 将针对 UAV-ground 视角差异、车辆可见部件不一致和文本属性粒度变化等问题，设计相应的跨视角视觉-语言对齐机制。

本文的主要贡献如下：
1. 提出 TAG-VR，一个文本增强的空地车辆检索 benchmark，面向 UAV 与地面视角之间的真实跨平台车辆检索问题。

2. 在 CVnet/CVPair 真实空地同 ID 数据基础上，补充 ID 级自然语言描述、图像级结构化属性和 QA 元数据，使数据集支持文本查询、属性检索和失败原因分析。同时，引入多个经过处理的空地感知数据集，抽取其中的空地车辆配对子集，通过车辆筛选、跨视角图像抽取、身份关联核验和统一文本标注，扩展 TAG-VR 的多源空地车辆样本。

3. 设计可审计的 VLM 辅助标注协议，约束文本标注聚焦跨视角稳定车辆属性，并显式处理小目标、遮挡、低置信度和背景泄漏风险。并建立覆盖 visual retrieval、text-to-image、image-to-text、text-guided cross-view retrieval、attribute/type retrieval 和跨数据源泛化评估的任务体系，为后续方法比较提供统一评估协议。
